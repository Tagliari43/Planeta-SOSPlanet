# Arquitetura e Engenharia (Pergaminhos Sagrados)

Este documento centraliza informações profundas sobre o "saber e o codificar" da SOSPlanet, permitindo que qualquer construtor navegue em nossos rios com sabedoria.

## 🧭 1. Camada Frontend (O Santuário / UI)

Nosso sistema interativo é concebido com um motor performático e moderno.

- **Mecânica Principal:** \`React\` em \`TypeScript\`. (Tipagem forte para honrar nossos contratos do código).
- **Moldura Estilizada:** \`TailwindCSS\` responsável pelo design "Mobile First", providenciando leveza visual, responsividade total, bordas neons futuristas e a atmosfera mágica de "Sci-Fi Mística" (Cyber-Regen).
- **Motion & Vida:** Todo fôlego no frontend é conferido pelas animações elásticas da biblioteca \`motion/react\` (Framer Motion).
- **Ícones do Planeta:** O oráculo visual é complementado por \`lucide-react\`.
- **Estatísticas (Oráculo):** Telas de governança paramétrica, oráculo de carbono, e fundos utilizam \`recharts\` para a geração de visualizações holográficas.

## 🔗 2. A Malha Descentralizada (Web3 Blockchain)

Nenhuma engrenagem atua no escuro. A arquitetura futura e a estrutura de dados são concebidas considerando que o balanço, os contratos de NFT Ecológico, a distribuição de recompensa e principalmente o \`PoA\` (Proof of Action - Prova pela Ação) são criptologicamente validados.

- As *Smart Contracts* (que processam a lógica de doações SOS) não estão localizadas diretamente nesta camada. Elas interagem com o sistema via bibliotecas de assinatura ou requisições seguras para oráculos parceiros.
- A **Tesouraria On-chain** utiliza componentes reativos na UI interligados com eventuais listeners de APIs e provedores da camada pública para que nenhum movimento financeiro perca visibilidade.

## 🌎 3. Impacto Operacional

Cada bloco da UI (\`/src/components\`) foi dividido em missões:
- \`PulsoVital.tsx\`: Representa a visão orgânica das métricas globais (Simulação / Telemetria On-chain de CO2).
- \`DiarioTesouro.tsx\`: Expõe informações para auditoria da transparência de gastos.
- \`MercadoRegen.tsx\`: Simula as trocas de ecossistemas locais.
